Sobre mí

Estudiante de Desarrollo de Aplicaciones Multiplataforma (DAM), especializado en el desarrollo de servicios back-end robustos y aplicaciones de escritorio con interfaces cuidadas.

Me muevo cómodo en todo el ciclo: desde el modelado de la base de datos y la lógica de negocio, hasta el despliegue containerizado del servicio en producción.

Durante mi FCT en InfoTelecomCloud trabajé con integración de APIs REST en entorno real, incluyendo gestión de autenticación por tokens y automatización de procesos.

Actualmente centrado en Java/Spring Boot para servicios web y Python/PySide6 para aplicaciones de escritorio, con Docker como pieza habitual del despliegue.

03

Control de tráfico de centralita · API 3CX

Integración API REST · FCT en InfoTelecomCloud

Proyecto desarrollado durante la FCT, para la gestión y monitorización del tráfico de una centralita telefónica a través de la API de 3CX. La API funciona con un access token que caduca cada 60 minutos, así que se automatizó su renovación cada 55 minutos y se añadió un botón de "forzar token" para regenerarlo manualmente si la conexión se corrompía antes de tiempo. Los endpoints se identificaron y probaron con Postman.

API REST3CXOAuth / TokensPostman
